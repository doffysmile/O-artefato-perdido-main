# <img src="https://skillicons.dev/icons?i=js" height="40" alt="JavaScript"/> O Artefato Perdido

Projeto desenvolvido utilizando **Construct** com implementação de lógica e mecânicas através de **JavaScript**, com foco em exploração, progressão e desenvolvimento de mecânicas interativas dentro de um ambiente de jogo 2D.

---

## Sobre o Projeto

**O Artefato Perdido** é um jogo desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento de jogos, aplicando lógica de programação e estruturação de eventos para criar uma experiência interativa focada na busca por um artefato perdido.

Durante o projeto foram trabalhados conceitos como movimentação, interação com objetos e progressão dentro do cenário.

---

## Tecnologias Utilizadas

* Construct
* JavaScript
* Lógica de Programação
* Desenvolvimento de Jogos 2D

---

## Funcionalidades

* Sistema de movimentação do personagem
* Interação com elementos do mapa
* Progressão por objetivos
* Sistema de exploração
* Estruturação de eventos e lógica do jogo

---

## Aprendizados

Este projeto contribuiu para o desenvolvimento de conhecimentos em:

* Programação baseada em eventos
* Estruturação lógica de sistemas interativos
* Desenvolvimento de mecânicas para jogos digitais
* Organização de projetos práticos
